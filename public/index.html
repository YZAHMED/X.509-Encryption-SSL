ProjectSSL — CryptoEncryption

Small demo: encrypt/decrypt using the server functions. If the API is not deployed, requests will return an error.

{ }
Curl examples 
# Basic GET
curl -i https://projectssl.netlify.app/

# Encrypt (Netlify functions expected at /api/encrypt)
curl -X POST https://projectssl.netlify.app/api/encrypt \
  -H "Content-Type: application/json" \
  -d '{"data":"hello world"}'

# Decrypt (Netlify functions expected at /api/decrypt)
curl -X POST https://projectssl.netlify.app/api/decrypt \
  -H "Content-Type: application/json" \
  -d '{"encryptedData":""}'
Tip: Netlify serves static files from the "publish" directory. This page expects any API to be deployed as Netlify Functions at /api/*. If you run locally, use http://localhost:3000/ or the port your server uses.
