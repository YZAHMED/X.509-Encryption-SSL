ProjectSSL — CryptoEncryption

X.509 certificates + RSA-OAEP encryption demo. API endpoints available at /api/encrypt and /api/decrypt

{ }
cURL examples: 
curl -i https://encryption.yaqoobahmed.com/
curl -X POST https://encryption.yaqoobahmed.com/api/encrypt \
  -H "Content-Type: application/json" \
  -d '{"data":"hello world"}'
curl -X POST https://encryption.yaqoobahmed.com/api/decrypt \
  -H "Content-Type: application/json" \
  -d '{"encryptedData":""}'
